# Anish Swarnakar - SAP UI5 Portfolio

Professional portfolio built with **SAP UI5 (OpenUI5)** and SAP Horizon theme.

## Features
- Hero Section with avatar and social links
- Stats Dashboard (CGPA, Projects, Certifications)
- About Me with contact info and languages
- Education timeline with cards
- Skills with progress indicators
- Projects with feature lists
- Achievements and Timeline
- Responsive design

## Run
```bash
cd portfolio
npm start
```
Opens at http://localhost:8080

## Structure
```
portfolio/webapp/
  index.html          - Entry point
  css/style.css       - Custom styles
  app/helpers.js      - Utility functions
  app/aboutTab.js     - About Me section
  app/educationTab.js - Education section
  app/skillsTab.js    - Skills section
  app/projectsTab.js  - Projects section
